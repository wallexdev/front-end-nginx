# Frontend com Nginx

Projeto de página frontend simples (Hello World) servida pelo **Nginx** em um **Ubuntu Server 24.04 LTS**.

## Tecnologias
- Ubuntu Server 24.04 LTS
- Nginx
- HTML

## Observações
Este servidor possui o Nextcloud instalado via Snap, que utiliza a porta 80.
Por isso, o Nginx foi configurado para rodar na porta **8080**.
